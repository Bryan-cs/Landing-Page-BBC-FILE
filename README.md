# BBC File — Landing Page

Sitio web oficial de **BBC File Service**, sistema de gestión de personal y seguridad social colombiana.

---

## ¿Qué es BBC File?

BBC File es una plataforma web diseñada para **outsourcing de nómina y consultoras de RRHH** en Colombia. Centraliza todo el ciclo de gestión de personal y seguridad social en un solo sistema: afiliaciones, seguimiento ARL, retiros, facturación con membrete y portal del cliente.

---

## Módulos del sistema

| # | Módulo | Descripción |
|---|--------|-------------|
| 01 | **Gestión de Afiliados** | Alta, baja y novedades por empresa. Control de EPS, AFP, ARL y CCF. |
| 02 | **Calculadora SS** | Cálculo automático de aportes según IBC y novedades vigentes. |
| 03 | **Facturación con Membrete** | Genera facturas para cada empresa cliente con el membrete de la consultora. |
| 04 | **Seguimiento ARL** | Control de accidentalidad, incapacidades y reportes ARL por trabajador. |
| 05 | **Cobro y Cartera** | Estados de cobro, historial de pagos y control de cartera por empresa. |
| 06 | **Retiros y Novedades** | Gestión de retiros, licencias e incapacidades con trazabilidad completa. |
| 07 | **Dashboard Financiero** | Resumen de ingresos, cartera, pendientes y proyecciones mensuales. |
| 08 | **Módulo de Tareas** | Asignación por usuario, alertas automáticas de vencimientos críticos. |
| 09 | **Portal del Cliente** | Acceso web para cada empresa: afiliados, facturas y reportes. |
| 10 | **Listas y Reportes** | Generación de reportes por período, exportación a Excel o PDF. |
| 11 | **Auditoría y Actividad** | Registro completo de cada acción: quién, cuándo y qué cambió. |
| 12 | **Backups Automáticos** | Copias de seguridad diarias en la nube. |
| 13 | **Seguimiento ARL avanzado** | Estadísticas de accidentalidad y seguimiento normativo. |
| 14 | **Portal Móvil (PWA)** | App instalable en iOS y Android, sin descarga de tienda. |
| 15 | **Control de Acceso** | Roles y permisos por usuario. |
| 16 | **Personalización** | Módulos adaptables al proceso específico de cada consultora. |

## Stack técnico

- **Frontend**: HTML5, CSS3 (custom properties), JavaScript vanilla
- **Tipografía**: Fraunces (display) + IBM Plex Sans (cuerpo) + IBM Plex Mono (datos)
- **Fuentes**: Google Fonts
- **Animaciones**: CSS keyframes + IntersectionObserver
- **PWA**: Progressive Web App ready
- **SEO**: Schema JSON-LD (SoftwareApplication, Organization, FAQPage), meta OG, sitemap.xml
